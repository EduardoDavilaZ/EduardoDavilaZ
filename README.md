# 👨‍💻 Sobre mí

Soy **Técnico Superior en Desarrollo de Aplicaciones Web** con interés por el desarrollo **Full Stack**, especialmente en backend, arquitectura de aplicaciones y diseño de interfaces modernas.

Durante mis prácticas en **THB Hotels** participé en el desarrollo de aplicaciones corporativas utilizando Laravel, JavaScript, MySQL e integración de APIs, colaborando en proyectos relacionados con automatización de procesos, análisis de datos e inteligencia artificial.
