# 👋 Sobre mí

Soy **Desarrollador de Aplicaciones Web** graduado en DAW.

Me apasiona desarrollar aplicaciones completas con especial interés en el **backend**, el diseño de **arquitecturas mantenibles**, las **bases de datos** y la creación de **interfaces modernas**.

Durante mis prácticas en **THB Hotels** participé en el desarrollo de herramientas internas utilizando **Laravel**, **JavaScript**, **MySQL** e integración de **APIs**, colaborando en proyectos relacionados con automatización, análisis de datos e inteligencia artificial.

Actualmente continúo ampliando mis conocimientos en tecnologías frontend modernas, Web Components y arquitectura de aplicaciones.

# 🚀 Proyectos destacados

## 📊 Analíticas OTA *(THB Hotels)*

Dashboard analítico desarrollado con Laravel para el seguimiento de reservas con **Price Match** y **Abandono de Carrito**, incorporando filtros dinámicos, gráficos interactivos, DataTables y consultas AJAX.

**Stack**

`Laravel` `JavaScript ES6` `AJAX` `DataTables` `MySQL`

---

## 🌱 Módulo Medioambiental (IA)

Aplicación destinada al análisis medioambiental hotelero mediante inteligencia artificial. Permite procesar facturas, gestionar prompts, generar informes automáticos y visualizar métricas energéticas.

**Stack**

`Laravel` `OpenAI API` `MySQL` `JavaScript`

---

## 🎓 Sistema de Gestión de Guardias
